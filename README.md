Defaults for projects
=====================


## Skills

Add github copilot rules to the .github folder.

```
cd .github
npx degit plumpNation/project-defaults/.github .
```

(skills-lock.json is not included in this)

## Typescript

```
cd <your-ts-project-root>
npx degit plumpNation/project-defaults/typescript .
```

## Python

```
cd <your-py-project-root>
npx degit plumpNation/project-defaults/python .
```
